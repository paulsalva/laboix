# LA BOIX Static HTML Desktop

Upload these files to the root of a GitHub repo and import it into Vercel as a static site.

Required structure:

```txt
index.html
style.css
script.js
assets/
  icons/
  images/
  video/
```

Artwork paths expected by the script:

```txt
assets/images/alla-grande.webp
assets/images/whatislove.webp
assets/images/baby-wants.webp
```

Replace placeholder links in `script.js`:

```txt
https://open.spotify.com/
https://bandcamp.com/
https://instagram.com/
https://www.tiktok.com/
https://youtube.com/
mailto:hello@example.com in index.html
```
